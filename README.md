The GUI application (Qt6) plays audio files (mp3, ogg, wav) with changeable pitch and speed.  


## Requirement

- Qt6  
- PyGI (PyGObject)  
- GStreamer 1.0 (Bad Plug-ins)  


## Usage

python pitchSpeedPlayer.py
